# 第二幕

## 三人でケルベロスに直接挑むが返り討ちにされる
場所 : ワシントンDC　ホワイトハウス近く

* 作戦を伝える
* 出雲の三種の神器を使う
  * 生大刀   : スクナヒコナが使う
  * 生弓     : イナバノシロウサギが使う
  * 天の詔琴 : オオクニヌシが使う。故事にならってケルベロスを眠らせるつもり。でもケルベロスと似てるだけで別の生き物だろうから効かないかも知れない。
* 天の詔琴（あめののりごと）は湿度が日本と違うのでひび割れて弦が切れて使えない
* 生大刀も生弓も出雲の国から離れているので神通力を発揮できなくて効かない
* ケルベロスの毒ガス攻撃！
  * スクナヒコナとイナバノシロウサギはかわす
  * オオクニヌシはまともに浴びて「ギャーーーー」
  * スクナヒコナが突っ込む。「ヌシ様はマスクかぶってるじゃないですか！」「そうだった」
* 撤退する

## 倒すための工夫が必要ということで怪獣召喚の方法を模索する
* 工夫しないと勝てないということで作戦会議
* やはり怪獣には怪獣をぶつけるのが良い

* エネルギーはBLMの民衆エネルギーから取得するとして、怪獣の種に困る

## 鵺を見つける
場所 : ワシントンDC上空

* 結局、アメリカに出た妖怪ってあの怪獣だったんすかね？と白ウサギが指摘する
* アメノガガミノフネの妖怪レーダーを動かすと、ホワイトハウスの庭にまだ妖怪がいるのが判る
  * ケルベロスは妖怪ではないらしい
* 大統領がBLMの掛け声で苛ついている
  * 声の出どころをカメラが追う
  * 行ってみると鵺である
* オオクニヌシ「妖怪の正体は鵺だったのか」
  * かつて京都に現われたという鵺も、その鳴き声で当時の公家たちを震え上がらせたという
  * 権力者の抱く怯えが鵺となり、その鵺が幻聴を聞かせるのだ
  * 1912年に日本から桜が贈られた際に一緒に渡ってきたのかも知れない
* そして、大統領が抱く幻聴へのいらだちがあのケルベロスに影響を与えてるのかも知れない
