# 第二幕

## 三人でケルベロスに直接挑むが返り討ちにされる
場所 : ワシントンDC　ホワイトハウス近く

* 作戦を伝える
* 出雲の三種の神器を使う
  * 生大刀   : スクナヒコナが使う
  * 生弓     : イナバノシロウサギが使う
  * 天の詔琴 : オオクニヌシが使う。故事にならってケルベロスを眠らせるつもり。でもケルベロスと似てるだけで別の生き物だろうから効かないかも知れない。
* 天の詔琴（あめののりごと）は湿度が日本と違うのでひび割れて弦が切れて使えない
* 生大刀も生弓も出雲の国から離れているので神通力を発揮できなくて効かない
* ケルベロスの毒ガス攻撃！
  * スクナヒコナとイナバノシロウサギはかわす
  * オオクニヌシはまともに浴びて「ギャーーーー」
  * スクナヒコナが突っ込む。「ヌシ様はマスクかぶってるじゃないですか！」「そうだった」
* 撤退する

## 倒すための工夫が必要ということで怪獣召喚の方法を模索する
* 工夫しないと勝てないということで作戦会議
* やはり怪獣には怪獣をぶつけるのが良い
* 怪獣を召喚したいが条件が厳しい
* エネルギーはBLMの民衆エネルギーから取得するとして、怪獣の種に困る

## 鵺を見つける
場所 : ワシントンDC上空

* アメリカに妖怪がいないと誰が決めた？と白ウサギが指摘する
* アメノガガミノフネの妖怪レーダーを動かすと、ホワイトハウスの庭に妖怪がいるのが判る
* 行ってみると鵺である
* 鳴き声が大統領を苛つかせている
* 鵺は、権力者の怯えが幻聴となり、その幻聴が鵺となると教えてくれる
* ケルベロスは鵺を探しているが鵺は勝ち目がないので逃げ回っている

## 鵺召喚
場所 : ホワイトハウスの庭

* オオクニヌシたちは鵺にケルベロス退治を手伝ってくれとお願いする
* 鵺も自分の身を守るためなので了承する
